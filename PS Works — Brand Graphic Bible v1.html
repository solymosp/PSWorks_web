TYPOGRAPHY

Type System

Primary sans-serif for all running text and headings. Monospace for data, code-like annotations, diagnostic language. No serif — this is not a law firm or a literary magazine.

VERSION 1 NOTES

Next Steps for Brand Development

PRIORITY 1
Logo refinement
Take Concept A or D to a designer. Build SVG mark, horizontal wordmark, and square icon version. Test at 16px and 200px.
PRIORITY 2
Typography decision
License Inter (free) or Neue Haas Grotesk (paid). Pair with JetBrains Mono for monospace elements. Set the type scale in CSS variables.
PRIORITY 3
Website component library
Build the failure mode card, diagnostic readout, aphorism block, and annotation style as reusable Framer or Webflow components.
PRIORITY 4
LinkedIn visual templates
Three Canva or Figma templates: aphorism post, failure mode analysis post, case study header. All using this colour system.
